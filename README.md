# Lapl4ce : Laplace Transform "Compiler"

Worked on this during semester 2 between ISA 2 because Unit 3 Maths (UE25MA141B) is built different. This C engine tokenizes mathematical equations, structures them into a pointer-linked Abstract Syntax Tree (AST) on the heap via a recursive descent parser, and generates $s$-domain algebraic strings using tree traversal.
